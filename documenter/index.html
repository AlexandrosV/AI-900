app.py

This Python code defines a Flask web application that provides an API for tracking and retrieving Bitcoin price data in EUR and CZK. When the application starts, it launches a background thread that periodically fetches the latest Bitcoin price from an external provider (configured via environment variables) using the get_price function, and stores the results in a database (presumably AWS DynamoDB) via the put_item function. The root endpoint (/) displays basic info about the app configuration. The /bitcoinday/<date> endpoint returns the average daily Bitcoin price for a given date (defaulting to today if not specified), while the /bitcoinmonth/<date> endpoint provides the average monthly price for a given month (defaulting to the current month if not specified). The endpoints return the average price in both EUR and CZK, along with the requested date and the time of the request. The application is intended to run as a standalone server and uses environment variables for configuration.
